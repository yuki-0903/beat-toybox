## Removed Docs

Old implementation plans and beatmap reports were removed because they described earlier prototypes:

- left/right lane movement only
- jump-over-obstacle gameplay
- obstacle avoidance as the main action
- temporary beatmap reports for specific generated files

The current MVP is now a 3-character rhythm performance runner.
