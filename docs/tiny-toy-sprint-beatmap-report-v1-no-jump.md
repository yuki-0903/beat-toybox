# Tiny Toy Sprint 譜面修正版メモ v1 / No Jump

## 変更内容

- ジャンプ操作を廃止
- 操作は左右移動のみ
- `jump_block` をすべて `block` に変換
- `Space` / タップジャンプ前提の記述を削除
- 3レーンのレーン移動だけで避けるリズムランナー譜面に調整

## 解析ベース

- 音源: `Tiny Toy Sprint.mp3`
- 長さ: 約 43.88 秒
- 推定BPM: 約 107.666
- レーン数: 3
- 障害物数: 73
- 変換したジャンプ障害物数: 5
- アイテム数: 6

## 操作仕様

### スマホ

- 左スワイプ: 左レーンへ移動
- 右スワイプ: 右レーンへ移動
- タップジャンプなし

### PC

- `A` / `←`: 左レーンへ移動
- `D` / `→`: 右レーンへ移動
- Spaceジャンプなし

## Codex向け注意

- `type: "jump_block"` は使わない。
- 実装側でもジャンプ判定・ジャンプ入力・空中状態は不要。
- 障害物はすべてレーン回避前提。
- プレイヤーは常に3レーンのどれかにいる。
- 音の気持ちよさは「移動成功音」「回避成功音」「コンボ音」で作る。

## おすすめのプレイ感

ジャンプをなくしたことで、ゲームの焦点はかなり明確になる。

> リズムに合わせて左右にスッスッと避けるだけ。

この方向ならスマホ片手プレイとの相性が良く、2.5D縦ランナーとしても実装しやすい。
